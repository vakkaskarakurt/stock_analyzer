# stock_analyzer
Analyzing price changes of turkish stocks in dollar terms.

Clone:
<pre>
<code>
git clone https://github.com/vakkaskarakurt/stock_analyzer
</code>
</pre>

Creating Enviroment:
<pre>
<code>
python -m venv stockenv
</code>
</pre>

Installing Requirements:
<pre>
<code>
pip install -r requirements.txt
</code>
</pre>

Run program:
<pre>
<code>
python main.py 
</code>
</pre>
